Sumary: Lattitude Vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled into from theOpenWeatherMap API to assemble a data set of over 500 cities.

After assembling the data said, we use matplotlib to plot various aspects of the weather versus latitude. factors will look at include: Temperature, clothing is, wind speed, humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations in the scriptures of any trends in correlations witnessed.
